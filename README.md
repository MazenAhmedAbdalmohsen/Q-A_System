## 📦 Repository Overview

This project allows for question answering over uploaded documents using:
- 📄 PDF and TXT file ingestion
- 💡 Google Gemini for LLM and Embeddings
- 🧠 LlamaIndex for retrieval
- 🌐 Streamlit for interactive UI
